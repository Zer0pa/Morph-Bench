# Data Policy

The staged repo may carry:

- docs and migration metadata
- neutral package code
- tiny synthetic fixtures

The staged repo may cite but should not yet vendor:

- Phase 4 authority artifacts from `workspace/artifacts/indus/phase4/`
- the cuneiform benchmark manifest family

The staged repo must not yet carry:

- raw corpora
- image-bearing benchmark payloads
- review-pack artifacts

Public promotion remains blocked until canonical license text is supplied and
the release boundary for image-bearing assets is written explicitly.
