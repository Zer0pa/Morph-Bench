# Gnosis Morph Bench

> Live synced methods scaffold for neutral benchmark and replay logic.
> Useful now, improving continuously.

## What This Repo Is

`gnosis-morph-bench` is the benchmark-first extraction target for the Gnosis
methods lane. It is supposed to own route comparison, permutation-null metrics,
stability and Jaccard checks, deterministic replay, and reference-freeze
helpers without pretending to own Indus or cuneiform scientific verdicts. This
staged repo is intentionally modest: it packages the source boundary, the
sovereign PRD, a corpus-neutral schema contract, and a runnable synthetic smoke
path while the full live-source extraction is still pending.
